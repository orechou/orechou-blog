Anthropic 家的 Claude 和 OpenAI 家的 ChatGPT 是目前最强的两个 AI 模型，但国内用户想用上它们的付费版，门槛不低——不支持国内支付、虚拟信用卡容易封号、各种渠道鱼龙混杂。
这篇文章把从注册到订阅的完整流程讲清楚，苹果用户有一条相对最稳的路。

---

## 一、先注册账号

### Claude

去 claude.ai 用 Gmail 注册，流程最标准。

几个要注意的点：

- 注册前用美国 IP，网络环境稳定，别频繁切换出口，新账号容易触发风控
- 用老 Gmail，别用新注册的邮箱
- 账号一人一号，别合租别共用

### ChatGPT

去 chatgpt.com 注册，同样建议用 Gmail。手机号验证那步，如果收不到验证码，可以用接码平台，但建议优先用自己的海外手机号，稳定很多。

---

## 二、订阅方案一览

**Claude 分三档：**

- Free：$0，基础对话，不含 Claude Code。
- Pro：$20/月，解锁 Claude Code，入门首选。适合普通用户，提供比免费版高出数倍的用量，通常每5小时可发送10-40个提示。
- Max：$100 或 $200/月，分别 5x 和 20x 用量，适合重度用户。专为对Claude依赖度极高的专业人士或深度用户设计。

**ChatGPT 分三档：**

- Free：$0，基础对话，也可使用 Codex。
- Plus：$20/月，适合普通深度用户。更高的 Codex 使用量，更复杂、更精准的图像生成。
- Pro：$100/月，适合重度用户。5 倍或 20 倍的使用配额。

---

## 三、最稳的付费方式：苹果内购

国内信用卡和 PayPal 都不好使，虚拟信用卡尤其是币圈渠道充值的秒封。最稳的方案是通过美区 App Store 内购——购买苹果礼品卡，充值到美区 Apple ID，然后在 App 里订阅。这条链路 Apple 自己背书，风险最低。

### 第 1 步：准备美区 Apple ID

如果你还没有美区 Apple ID，按这个流程注册：

1. 打开一个美国地址生成器（推荐 meiguodizhi.com，选蒙大拿州等免税州），生成一个美国地址和电话
2. 前往苹果官网（appleid.apple.com/account）注册 Apple ID，全程挂美国节点
3. 国家选美国，邮箱和手机号可以用大陆的，填完验证码即可
4. 注册完后进入账户管理，点「个人信息」→「国家或地区」，国家选美国
5. 付款方式选「无」，下面填入刚才生成的美国地址和电话。如果提示电话号码无效，换个地址再试。这个号码只是账单地址，不需要真实有效
6. 保存后这个 ID 就可以在 App Store 正常使用了

注意：只在 App Store 里登录美区 ID，**不要在系统设置的 Apple ID 里登录**，否则会影响你国区账号的正常使用。然后下载 Claude 和 ChatGPT 两个应用。

### 第 2 步：购买苹果礼品卡

这一步是通过第三方平台购买美国区苹果礼品卡，然后用兑换码给美区 Apple ID 充值。具体流程：

1. 在支付宝中切换定位，例如选择旧金山
2. 在支付宝中打开 Pockyt（或其他类似的苹果礼品卡购买平台），在 App 内选择美国区苹果礼品卡，选择需要的面额下单。
3. 支付宝支付，按照平台的流程完成付款。下单后平台会将礼品卡兑换码发送到你的账号中，通常几分钟到一小时内到账。
4. 拿到兑换码后，在 iPhone 的 App Store 里登录美区 Apple ID（注意不要在系统设置里登录，只在 App Store 里登录即可）。点击右上角头像，选择「兑换充值卡或代码」，把兑换码输入进去。充值成功后可以在账户界面看到余额。

如果选择的面额刚好覆盖订阅费用，直接用余额支付就行。iOS 端订阅 App Store 会根据你的账单地址所在州收取税费，免税州选对了可以免去这部分。

### 第 3 步：订阅

充值完成后，打开对应的 App：

**Claude：** 打开 Claude App → 选 Pro 订阅 → 弹出 Apple ID 支付确认 → 用余额支付

**ChatGPT：** 打开 ChatGPT App → 点 Plus 订阅 → 弹出 Apple ID 支付确认 → 用余额支付

订阅成功后，这个付费状态跟账号走。你在 iOS 订完，Android 或网页端登录同一个账号，付费权益照样生效。

取消订阅在 App Store → 点头像 → 订阅 里操作。Claude 和 ChatGPT 都是如此。

---

## 四、保号建议

账号封了什么都得重来，订阅前几件事值得注意：

- **网络环境**：用稳定低延迟的线路，别天天换节点
- **一号一人**：别合租别共用，尤其是 Max 和 Plus 这类高价值订阅
- **付款方式**：走 Apple 内购或实体卡，虚拟卡尤其是币圈渠道充值的容易封
- **邮箱**：老 Gmail 优先，新注册的邮箱风控概率高
- **出口干净**：别让乱七八糟的 App 流量都从同一个代理出口走

---

整个过程看起来步骤不少，但实际操作一遍大概 20 分钟就能搞定。最关键的两件事：用一个靠谱的美区 Apple ID，走苹果官方礼品卡充值。这两步做对了，后面就是点点屏幕的事。
